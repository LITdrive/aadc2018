This archive contains different variants of an OpenDrive file created by atlatec.

KA-Suedtangente-Vires.xodr is optimized for Vires tools. 
KA-Suedtangente-atlatec.xodr is for general use, without using the RoadShape feature of OpenDrive.
KA-Suedtangente-atlatec-RoadShape.xodr is for general use if the OpenDrive interpreter supports RoadShape.

More information can be found in KA-Suedtangente-Xodr-Vires.pdf, containing screenshots of the Vires variant.
